for running the website please follow the below steps.

1) Extract all laravel folders to htdocs  in your xampp folder and copy the "titl2.jpg" to "htdocs/img" in your xampp folder.
2) create a database with name register (it's mandatory) in your xampp/phpmyadmin.
3)now select the database register and click on import and then select the register.sql file.
4)click go.
5)now start Apache and mysql in your xampp controll panel.
6)open your web browser and type "localhost/laravel".
7)The website will open.
